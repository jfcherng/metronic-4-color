# Install via `yarn` (or `npm`)

```bash
yarn add metronic-4-color
```

```bash
npm install metronic-4-color --save
```


# To use

- The source files are under the `src/` directory.
- The built files are under the `dist/` directory.


# To build

```bash
yarn build-full
```

```bash
npm build-full
```
